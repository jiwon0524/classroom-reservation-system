# Classroom Reservation System Evidence Seed

This local seed project provides the first executable evidence for the classroom reservation system.

## Current Scope

- Requirement covered: `STR-008`
- Rule: classroom reservations must not exceed 50% of classroom capacity.
- Test framework: JUnit 5 console launcher

## Notes

- The course requirement is Java 21, but this workspace currently exposes Java 17 on `PATH`.
- These tests are intentionally small and should be moved into the real project repository once the team creates it.
- GitHub PR evidence still requires a real GitHub repository remote.
